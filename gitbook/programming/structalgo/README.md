## 자료구조

- 컴퓨터 과학에서 효율적인 접근 및 수정을 효과적으로 하기 위한 자료의 조직, 관리, 저장 체계
- 데이터 값의 모임, 데이터간의 관계, 데이터에 적용할 수 있는 함수나 명령을 의미
- **메모리를 효율적으로 사용**하기 위해 적절한 자료구조 선택이 중요하다.
- **프로그램 설계 시 가장 먼저 고려**되어야 한다.
- 프로그램의 **성능에 직접적인 영향**을 미친다.
