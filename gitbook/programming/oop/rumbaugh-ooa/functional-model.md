## 기능 모델의 장점

1. 데이터 흐름을 명확하게 분석 가능하다.
2. 시스템 기능을 시각적으로 표현 할 수 있다.
3. 실제 시스템 개발 시 기능 정의에 유용하다.
4. 객체 모델 및 동적 모델과 결합하여 완전한 시스템 설계가 가능하다.
