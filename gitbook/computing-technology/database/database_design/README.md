# 데이터베이스 설계

## 데이터베이스 설계란?

* 효율적인 데이터 저장, 검색, 관리를 위해 데이터베이스 구조를 정의하는 과정을 의미한다.
* 설계가 잘못되면 데이터 중복, 성능 저하, 유지보수 문제가 발생할 수 있기 때문에 체계적인 접근이 필요하다.

***
