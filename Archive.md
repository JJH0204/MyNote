압축과 다른 방법으로 한 덩어리로 묶는 기능

# 압축
확장자: .xz .bz2 .gz .zip 등등
툴 : xz, bzip2, gzip
명령어: 
- `xz 파일명` : 압축
- `xz -d 파일명.xz` : 압축 해제
특징:
- 원본 파일이 남지 않는다.
- 압축이 꼭 용량이 줄어든 다는 것은 아니다
- 압축 툴에 따라 효율이 다르다.

# 아카이브
명령어: 
	- `tar [옵션] [아카이브이름] [파일 또는 디렉토리]`: 
옵션:
	- c : 아카이브를 만들어라
	- f : 대상 파일을
	- x : 아카이브 해제
	- v : 과정 출력
	- J : 아카이브 후 .xz 압축을 수행한다.
	- j : 아카이브 후 .bz2 압축 수행
	- z : 아카이브 후 gzip  압축 수행
특징:
- 원본 파일이 남는다. (재활용이 가능)
