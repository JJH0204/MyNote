# 어셈블 (Assemble)
---
> 컴파일로 생성된 어셈블리어 코드를 [ELF](ELF.md)형식의 목적 파일(Object file)로 변환하는 과정
> 윈도우에서 어셈블한다면 목적 파일은 PE 형식을 갖게 된다.
> 목적 파일로 변환되면 기계어로 번역됨으로 더 이상 사람이 해석하기 어려워 진다.

다음은 gcc의 -c 옵션으로 add.s 파일을 목적 파일로 변환하고, 결과로 나온 파일을 16진수로 출력한것이다.

$ gcc -c add.S -o add.o
$ file add.o
