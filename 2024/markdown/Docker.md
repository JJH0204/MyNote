# 도커를 사용하는 이유
---
- 가상 머신을 설치하여 직접 문제 환경을 구축할 필요 없이 도커 이미지를 선택해 컨테이너를 구축하는 것으로 한번에 해결이 가능하다.
- 직접 환경을 구축하다 보면 의도와 다른 환경 설정이 포함될 수도 있기 때문에 [변인통제](https://ko.wikipedia.org/wiki/%EB%B3%80%EC%9D%B8%ED%86%B5%EC%A0%9C)가 불가능하다.
