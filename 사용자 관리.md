# /etc/passwd
---
```
[root@localhost jaeho]# cat /etc/passwd
...
jaeho:x:1000:1000:jaeho:/home/jaeho:/bin/bash
vboxadd:x:976:1::/var/run/vboxadd:/bin/false
```

jaeho: 사용자 이름
x: 암호화된 사용자 비밀번호
1000: 사용자 id
1000: 그룹 id
jaeho: 그룹 이름
/home/jaeho: 홈 디렉토리 경로
/bin/bash: 기본 쉘(shell)

# /etc/group
---
```
[root@localhost jaeho]# cat /etc/group
...
jaeho:x:1000:
```

jaeho: 그룹이름
x: 암호화된 그룹 비밀번호
1000: 그룹 id
(공백): 소속 인원
