# 모의 해킹 진행 과정
1. client -> 보안업체: 모의해킹 의뢰

3. 모의 해킹 후 결과 보고

# 모의 해킹 방식
1. 화이트 박스
	- 모의 해킹 의뢰자(클라이언트)로 부터 정보를 받아서 진행하는 모의 해킹
	- 보안 담장자가 모르는 의외의 취약점을 발견할 확률이 낮다.
1. 그레이 박스
	- 화이트 박스와 블랙 박스의 중간
2. 블랙 박스
	- 모의 해킹 의뢰자(클라이언트)로 부터 정보를 받지 않고 진행하는 모의 해킹
	- 보안 담당자가 모르는 의외의 취약점을 발견할 확률이 높다.
	- 난이도가 높은 만큼 작업 기간이 길어지고 비용이 비싸진다.
