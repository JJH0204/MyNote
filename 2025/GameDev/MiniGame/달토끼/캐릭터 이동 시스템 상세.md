```csharp
private void DirectionInput()
    {
        // 수평 입력 처리
        var horizontal = Input.GetAxis("Horizontal");

        // 방향 벡터 생성
        var direction = new Vector2(horizontal, 0).normalized;

        // 방향 벡터가 유효한 경우에만 처리
        if (direction.magnitude >= 0.1f)
        {
            // 방향 벡터를 사용하여 플레이어 객체의 방향 전환
            if (direction.x < 0)
            {
                _currentDirection = Direction.Left;
                GameManager.Instance.Player.GetComponent<Player>().SetDirection(Direction.Left);
            }
            else if (direction.x > 0)
            {
                _currentDirection = Direction.Right;
                GameManager.Instance.Player.GetComponent<Player>().SetDirection(Direction.Right);
            }
        }
        else
        {
            _currentDirection = Direction.Center;
        }
    }
```

```md
⸻

🎈 탄성이란?

**탄성(elasticity)**은 어떤 물체가 힘을 받아 변형되었을 때, 원래 모양으로 돌아가려는 성질입니다.
	•	예: 고무공을 땅에 던지면 튀어 오르죠? → 이것이 탄성입니다.
	•	반대로 진흙이나 찰흙은 변형되면 돌아오지 않죠 → 비탄성입니다.

⸻

📐 핵심 개념 요약 (게임 물리에 필요한 부분만)

개념	설명	예시
탄성 계수 (Coefficient of Restitution)	두 물체가 충돌했을 때, 충돌 후 얼마나 튕겨 나오는지를 나타내는 수치 (0~1 사이)	1: 완전 탄성, 0: 완전 비탄성
반발력 (Bounciness)	Unity에서는 PhysicsMaterial2D.bounciness로 구현됨. 충돌 후 얼마나 튕기는지를 결정	고무공: 0.8 / 축구공: 0.6 / 돌: 0.2


⸻

🎮 Unity에서의 탄성 구현

Unity에서 탄성을 구현하려면 다음 2가지를 설정하면 됩니다:

1. Physics Material 2D 생성
	•	Assets > Create > Physics Material 2D
	•	Bounciness: 0.0 ~ 1.0 (얼마나 튀길지)
	•	Friction: 마찰력

2. Collider에 Material 적용
	•	CircleCollider2D, BoxCollider2D 등 선택
	•	Material에 위에서 만든 Physics Material 2D 적용

⸻

💡 직관적 요약

물체	Bounciness	느낌
고무공	0.9	잘 튐
농구공	0.6	약간 튐
나무 블록	0.2	거의 안 튐
진흙 덩어리	0	전혀 안 튐


⸻

📘 더 알고 싶다면
	•	탄성 충돌 vs 비탄성 충돌
	•	운동 에너지 보존 여부
	•	Hooke’s Law (고급 개념: 스프링 같은 것에 적용)
```
