# RPM(Redhat package manager)
---
- rpm 옵션 대상파일(프로그램)
	- -U: 업데이트 없으면 다운로드
	- -v: 다운로드 과정 로그 출력
	- -h: 다운로드 과정 상세 출력
	- -e: 삭제
	- -qa: rpm 툴로 설치 여부 확인
	- Failed dependencies
		- RPM 방식으로 설치 시 의존성 문제가 발생할 수 있다.
- yum = dnf
	- dnf install ~
	- dnf localinstall ~
	- dnf list ~
	- dnf remove ~
	- dnf grouplist
