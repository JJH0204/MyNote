**IP 패킷의 TCP/UDP 포트 숫자와 소스 및 목적지의 IP 주소 등을 재기록하면서 라우터를 통해 네트워크 트래픽을 주고 받는 기술을 말한다.**

### Network_Address_Translation(이하 NAT)를 이용하는 이유
- 사설 네트워크에 속한 여러 개의 호스트가 하나의 공인 IP 주소를 사용하여 인터넷에 접속하기 위함
- IPv4의 주소 부족 문제
- 비공인(사설, local) 네트워크 주소를 사용하는 망에서 외부 공인망(public, eo

호스트 간의 통신에 있어서 복잡성을 증가시킬 수 있으므로 네트워크 성능에 영향을 줄 수 있다
