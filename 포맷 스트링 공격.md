## 예제 코드
```c
#include <stdio.h>

{

}
```

### 포맷 스트링을 활용한 메모리 주소 검출
```c
#include <stdio.h>

  

int main()

{

char *buffer = "jaeho\n%x\n"; // %x insert

  

printf(buffer);

  

return 0;

}
```
