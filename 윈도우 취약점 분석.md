135/tcp - msrpc (Microsoft Windows RPC)
- **서비스**: Microsoft 원격 프로시저 호출(RPC) 서비스.
- **설명**: RPC는 분산 네트워크 환경에서 원격으로 프로세스를 호출하는 데 사용되며, 다양한 Windows 서비스가 RPC에 의존해 작동해.
- **공격 포인트**: 과거 **MS03-039**와 같은 취약점이 RPC 서비스에 있었으며, 이를 통해 공격자가 원격 코드 실행을 할 수 있었어. RPC는 특히 Windows에서 자주 표적이 되며, 최신 보안 패치가 적용되지 않은 경우 권한 상승 공격이 가능해.
**139/tcp - netbios-ssn (NetBIOS Session Service)**
- **서비스**: NetBIOS 세션 서비스로 파일 및 프린터 공유 기능을 제공
- **설명**: 일반적으로 SMB(445번 포트)와 함께 작동하며, 오래된 버전의 SMB 프로토콜을 지원하는 시스템에서 취약점이 발생할 수 있어.
- **공격 포인트**: **SMB 리버스 셸**을 통한 원격 코드 실행 또는 **NetBIOS 이름 서비스 스푸핑 공격** 등을 시도할 수 있어. 또한, 무차별 대입 공격을 통해 공유 폴더에 접근할 수 있어.
**445/tcp - microsoft-ds (SMB)**
- **서비스**: Microsoft-DS(SMB) 서비스.
- **설명**: SMB는 파일 공유 및 프린터 공유를 위한 프로토콜로, 특히 Windows 환경에서 많이 사용돼.
- **공격 포인트**: SMB는 **EternalBlue(CVE-2017-0144)**와 같은 취약점이 있어 이를 통해 **원격 코드 실행**이 가능할 수 있어. Metasploit과 같은 도구에서 **smb_login** 모듈을 통해 무차별 대입 공격을 시도하거나, 패치되지 않은 경우 **권한 상승** 공격이 가능해.
**5985/tcp - http (Microsoft HTTPAPI httpd 2.0 - SSDP/UPnP)**
- **서비스**: Microsoft의 HTTP API로 Windows 원격 관리(WinRM) 기능과 관련이 있음.
- **설명**: 이 포트는 주로 WinRM을 통해 **원격 명령 실행**을 가능하게 하며, Windows 관리자가 원격 시스템을 관리하는 데 사용돼.
- **공격 포인트**: WinRM을 통해 PowerShell 명령을 원격에서 실행할 수 있으며, 자격 증명이 있는 경우 이 포트를 통해 **원격 명령 실행**을 시도할 수 있어. Metasploit에서 WinRM 모듈을 활용해 공격할 수 있음.
**49155/tcp - msrpc (Microsoft Windows RPC)**
- **서비스**: 고정된 포트 대신 **동적 포트**에서 RPC 서비스를 제공.
- **설명**: 동적 RPC 서비스는 요청이 있을 때마다 포트를 할당해 사용하기 때문에 다양한 포트에서 RPC 서비스가 열릴 수 있어.
- **공격 포인트**: 다른 RPC 포트와 유사하게, 권한 상승이나 원격 코드 실행 취약점을 노려볼 수 있어.
