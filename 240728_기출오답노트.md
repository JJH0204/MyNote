**문제**: 다음 중 ping 명령어와 가장 관련이 깊은 프로토콜로 알맞은 것은?

1. TCP
2. UDP
3. ARP
4. ICMP

**정답**: 4. ICMP

해설:
TCP(Transmission Control Protocol): 신뢰성 있는 데이터 전송을 보장하는 프로토콜, 연결 지향적
UDP(User Datagram Protocol): 비연결형 프로토콜, 신뢰성보다는 속도가 중요한 경우에 사용
ARP(Address Resolution Protocol): IP 주소를 물리적 네트워크 주소(MAC 주소)로 변환하는데 사용
ICMP(Internet Control Message Protocol): 네트워크 진단과 에러 메시지 전송하는데 사용

**문제**: 다음 중 논리 파티션에 부여될 수 있는 최소 파티션 값으로 알맞은 것은?

1. 1
2. 2
3. 4
4. 5

**정답**: 4. 5
